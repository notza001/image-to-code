# Component Patterns — พื้นฐานที่ต้องรู้

> ก่อนแปลงเป็นโค้ด ต้องรู้จัก component ก่อน

---

## 1. Button (ปุ่ม)

### รูปทรง
- สี่เหลี่ยมมุมโค้ง (rounded rectangle)
- ขนาดเล็ก กดได้

### รูปแบบ
```
┌─────────────┐
│    Text     │
└─────────────┘

┌─────────────┐
│ Icon  Text  │
└─────────────┘
```

### ส่วนประกอบ
- Background (พื้นหลัง)
- Text (ข้อความ)
- Border (ขอบ) — อาจมีหรือไม่มี
- Icon (ไอคอน) — อาจมีหรือไม่มี

### Variants
| Variant | ลักษณะ |
|---------|--------|
| Primary | สีหลัก, เด่นที่สุด |
| Secondary | สีรอง |
| Outline | มีขอบ, พื้นโปร่ง |
| Ghost | ไม่มีพื้น ไม่มีขอบ |

### States
| State | ลักษณะ |
|-------|--------|
| Default/Active | ปกติ |
| Hover | เมื่อ mouse ชี้ |
| Focus | เมื่อ focus |
| Disabled | ปิดการใช้งาน (สีจาง) |
| Loading | กำลังโหลด |

### Sizes
- Small (sm)
- Medium (md) — default
- Large (lg)

---

## 2. Badge (ป้าย)

### รูปทรง
- สี่เหลี่ยมมุมโค้งมาก หรือ วงรี
- ขนาดเล็กมาก

### รูปแบบ
```
┌───────┐
│ Text  │
└───────┘
```

### ส่วนประกอบ
- Background
- Text
- อาจมี icon เล็กๆ

### Variants
| Variant | ใช้เมื่อ |
|---------|---------|
| Success | สำเร็จ (เขียว) |
| Warning | เตือน (เหลือง) |
| Error | ผิดพลาด (แดง) |
| Info | ข้อมูล (น้ำเงิน) |
| Neutral | ทั่วไป (เทา) |

---

## 3. Input (ช่องกรอกข้อมูล)

### รูปทรง
- สี่เหลี่ยมมุมโค้งเล็กน้อย
- มีขอบ
- ยาวกว่าสูง

### รูปแบบ
```
Label
┌─────────────────────┐
│ Placeholder...      │
└─────────────────────┘
Helper text

┌─────────────────────┐
│ Icon │ Placeholder  │
└─────────────────────┘
```

### ส่วนประกอบ
- Label (ป้ายชื่อ) — ด้านบน
- Input field (ช่องกรอก)
- Placeholder (ข้อความจาง)
- Helper text (ข้อความช่วย) — ด้านล่าง
- Icon — อาจมี ซ้ายหรือขวา

### States
| State | ลักษณะ |
|-------|--------|
| Default | ขอบเทา |
| Focus | ขอบสีหลัก |
| Error | ขอบแดง |
| Disabled | พื้นเทา |

---

## 4. Card (การ์ด)

### รูปทรง
- สี่เหลี่ยมมุมโค้ง
- ขนาดใหญ่กว่า button
- มี shadow หรือ border

### รูปแบบ
```
┌─────────────────────┐
│      Image          │
├─────────────────────┤
│ Title               │
│ Description         │
│                     │
│ [Button]            │
└─────────────────────┘
```

### ส่วนประกอบ
- Image (รูป) — อาจมีหรือไม่มี
- Title (หัวข้อ)
- Description (คำอธิบาย)
- Actions (ปุ่ม) — อาจมีหรือไม่มี
- Meta info (ข้อมูลเพิ่ม) — วันที่, ผู้เขียน

---

## 5. Navbar (แถบนำทาง)

### รูปทรง
- แถบยาวแนวนอน
- อยู่ด้านบนสุดของหน้า

### รูปแบบ
```
┌─────────────────────────────────────────┐
│ Logo    Menu1  Menu2  Menu3    [Button] │
└─────────────────────────────────────────┘
```

### ส่วนประกอบ
- Logo (ซ้าย)
- Menu items (กลาง)
- Actions — ปุ่ม, user avatar (ขวา)

---

## 6. Modal (หน้าต่างลอย)

### รูปทรง
- สี่เหลี่ยมมุมโค้ง
- ลอยกลางจอ
- มี overlay ด้านหลัง

### รูปแบบ
```
┌─────────────────────┐
│ Title            X  │
├─────────────────────┤
│                     │
│ Content             │
│                     │
├─────────────────────┤
│      [Cancel] [OK]  │
└─────────────────────┘
```

### ส่วนประกอบ
- Header (Title + Close button)
- Body (Content)
- Footer (Actions)

---

## 7. Table (ตาราง)

### รูปทรง
- ตารางแถวและคอลัมน์
- มี header แถวแรก

### รูปแบบ
```
┌────────┬────────┬────────┐
│ Head 1 │ Head 2 │ Head 3 │
├────────┼────────┼────────┤
│ Data   │ Data   │ Data   │
├────────┼────────┼────────┤
│ Data   │ Data   │ Data   │
└────────┴────────┴────────┘
```

---

## 8. Avatar (รูปประจำตัว)

### รูปทรง
- วงกลม หรือ สี่เหลี่ยมมุมโค้ง
- ขนาดเล็ก

### รูปแบบ
```
 ┌───┐
 │ A │  ← ตัวอักษร (ถ้าไม่มีรูป)
 └───┘

 ┌───┐
 │ 🖼 │  ← รูปภาพ
 └───┘
```

### Sizes
- xs, sm, md, lg, xl

---

## 9. Checkbox / Radio

### รูปทรง
- Checkbox: สี่เหลี่ยมเล็ก
- Radio: วงกลมเล็ก

### รูปแบบ
```
☐ Label     ← Checkbox (unchecked)
☑ Label     ← Checkbox (checked)

○ Label     ← Radio (unchecked)
● Label     ← Radio (checked)
```

---

## 10. Select / Dropdown

### รูปทรง
- คล้าย Input แต่มีลูกศรขวา
- กดแล้วมี menu ลอยออกมา

### รูปแบบ
```
┌─────────────────────┐
│ Selected      ▼     │
└─────────────────────┘
        ↓ (เมื่อกด)
┌─────────────────────┐
│ Option 1            │
│ Option 2            │
│ Option 3            │
└─────────────────────┘
```

---

## สรุป: วิธีระบุ Component

1. **ดูรูปทรง** — สี่เหลี่ยม? วงกลม? แถบยาว?
2. **ดูขนาด** — เล็ก (Button/Badge)? กลาง (Card)? ยาว (Navbar)?
3. **ดูส่วนประกอบ** — มีรูป? มี text? มีปุ่ม?
4. **ดูตำแหน่ง** — บนสุด (Navbar)? ลอย (Modal)?
